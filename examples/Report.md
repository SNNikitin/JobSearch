## 3. EXPERIENCE — Детальный анализ

### Проблема #1: ОПИСАНИЯ — СПИСОК ЗАДАЧ, А НЕ ДОСТИЖЕНИЙ

Почти все позиции описаны в формате "что делал" вместо "чего достиг":

**Текущий стиль (плохо):**
> - Designing and developing software for iOS / Android using React Native / TypeScript
> - Using Git for version control
> - Conducting code reviews

**Нужный стиль (хорошо):**
> - Spearheaded migration from JavaScript/Flow to TypeScript + Zustand, reducing runtime errors by 40% and improving developer onboarding time
> - Architected and shipped a cross-platform mobile banking app serving 50K+ active users
> - Established CI/CD pipeline with Fastlane + GitHub Actions, reducing release cycle from 2 weeks to 2 days

### Проблема #2: МНОЖЕСТВО КОРОТКИХ КОНТРАКТОВ

| Позиция | Длительность | Как выглядит |
|---------|-------------|--------------|
| Mercury Development | 3 месяца | Красный флаг |
| Goglia Nutrition | 4 месяца | Красный флаг |
| Self-employed | 7 месяцев | Нормально (фриланс) |
| Arcadia Global Solutions | 7 месяцев | ОК (отмечено как temp contract) |
| InGames.io | 8 месяцев | Жёлтый флаг |
| TalentTech | 10 месяцев | Жёлтый флаг |

**Решение:** Для коротких контрактов ВСЕГДА указывать "(Contract)" или "(Temporary project)" в названии позиции. Это нормализует восприятие — рекрутеры понимают контрактную работу.

### Проблема #3: ПОТЕРЯ СЕНЬОРНОСТИ В ТИТУЛАХ

| Период | Компания | Титул | Проблема |
|--------|---------|-------|----------|
| 2019–2022 | Mango Telecom | **Team Lead** | Отлично |
| 2022–2023 | Qlean.ru | **Team Lead** | Отлично |
| 2022–2023 | InGames.io | Senior Mobile Developer | ОК |
| 2023 | Mercury | **Senior** RN Developer | ОК |
| 2023–2025 | Nitka Technology | **Mobile Developer** | ДАУНГРЕЙД! |
| 2025 | Arcadia | **RN Developer** | ДАУНГРЕЙД! |

Вы были **Team Lead** в двух компаниях, а последние 2 позиции — просто "Developer" без "Senior". Это выглядит как деградация карьеры.

**Решение:** Если реально выполняли senior-обязанности — обсудите с бывшими работодателями корректировку титула, или добавьте уточнение в описание: "Performed senior-level responsibilities including..."

### Проблема #4: ТЕКУЩИЙ ГЭП В ЗАНЯТОСТИ

Последняя позиция закончилась в ноябре 2025. Сейчас февраль 2026 — это **3+ месяцев** без работы. На LinkedIn это видно и может настораживать.

**Решение:** Добавить текущую активность:
- "Freelance React Native Consultant (December 2025 — Present)" если есть любые проекты
- Или указать "Career break — upskilling in React Native New Architecture, Expo ecosystem"
- Или активный Open Source вклад

## 6. ОЦЕНКА ШАНСОВ БЫТЬ ЗАМЕЧЕННЫМ

### LinkedIn Search Algorithm (Recruiter Lite)

Рекрутеры ищут по ключевым словам. Алгоритм индексирует:
1. **Headline** — вес x3
2. **About** — вес x2
3. **Experience titles** — вес x2
4. **Skills** — вес x1.5
5. **Experience descriptions** — вес x1

Типичный поисковый запрос рекрутера:
```
"React Native" AND "TypeScript" AND "Senior" AND "Remote"
```

### Ваш профиль в этом поиске:

| Слово | Headline | About | Titles | Skills | Описания |
|-------|----------|-------|--------|--------|----------|
| React Native | ✅ | ❌ нет About | ✅ (3 из 10) | ✅ | ✅ |
| TypeScript | ❌ | ❌ | ❌ | ✅ | ✅ (4 раза) |
| Senior | ✅ | ❌ | ✅ (3 из 10) | ❌ | ❌ |
| Remote | ❌ | ❌ | ❌ | ❌ | ❌ |
| Jest | ❌ | ❌ | ❌ | ❌ | ❌ |
| Redux | ❌ | ❌ | ❌ | ❌ | ❌ |
| Zustand | ❌ | ❌ | ❌ | ❌ | ✅ (1 раз) |

### Прогноз видимости:

```
Вероятность появиться в ТОП-20 поисковой выдачи:  ~10-15%
Вероятность, что рекрутер кликнет на профиль:     ~20-30%
Вероятность, что рекрутер напишет сообщение:       ~5-10%
Вероятность полного funnel (найден→контакт):       ~1-3%
```

С оптимизированным профилем эти цифры можно поднять до **15-25%** полного funnel.
