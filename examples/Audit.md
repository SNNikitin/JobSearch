### Описание роли (типичное)

Мы ищем Senior React Native Developer для разработки и поддержки
кроссплатформенного мобильного приложения. Вы будете проектировать
архитектуру, менторить команду, участвовать в code review и принимать
технические решения. Ожидается end-to-end ownership фич — от
проектирования до релиза и мониторинга.

## 8. SENIOR vs MID-LEVEL — Что отличает

| Аспект | Mid-Level | Senior |
|--------|-----------|--------|
| Архитектура | Следует паттернам | **Проектирует архитектуру**, пишет ADR |
| Native код | Использует модули | **Пишет и дебажит** нативные модули |
| New Architecture | Знает о ней | **Мигрировал приложения**, ведёт команду |
| Перформанс | Фиксит очевидное | **Профилирует системно** — Xcode Instruments, Hermes bytecode |
| Тестирование | Пишет юниты | **Проектирует стратегию** (пирамида unit/integration/E2E) |
| CI/CD | Использует пайплайны | **Настраивает и поддерживает** end-to-end |
| Scope | Фичи | **Product area или всё приложение** |
| Оценка | Свои задачи | **Для команды**, выявляет риски |
| Security | Базовое понимание | **Реализует** pinning, secure storage, obfuscation |

## 10. РЕКОМЕНДУЕМЫЙ ПЛАН ДЕЙСТВИЙ

### Немедленно
1. Убедиться в advanced TypeScript (generics, strict mode, utility types)
2. Получить hands-on опыт с **New Architecture** (Fabric, TurboModules, JSI)
3. Глубоко изучить **Expo экосистему** (EAS Build/Update, Router, Modules API)
4. Освоить **Zustand + TanStack Query** (помимо Redux Toolkit)
5. Подготовить **system design** для мобильных приложений

### В течение 3 месяцев
6. **Reanimated 3** + Gesture Handler
7. **FlashList**, **MMKV** и современные performance-паттерны
8. CI/CD пайплайн на **EAS Build + GitHub Actions**
9. Усилить **native debugging** (Xcode Instruments, Android Profiler)
10. Подготовить talking points по **архитектурным решениям**

### В течение 6 месяцев
11. Вклад в **open-source RN библиотеку**
12. Базовый **Swift** и **Kotlin** для нативных модулей
13. **React Native Web** (universal apps)
14. Портфолио-проект: New Architecture + Expo
