# Промпты и примеры ответов по улучшению резюме

## 1. Анализ существующих вакансий, построение усредненной вакансии
### Prompt:
```
Проведи глубокий поиск и анализ вакансий {ВАКАНСИЯ} на LinkedIn.com. 
Собери список запрашиваемых в вакансиях скиллов, как хард, так и софт.
Отсортируй их список по важности и частотности.
Отдельно составь список скиллов из категории "расширенные пожелания".
В итоге должна получиться усредненная "вакансия-шаблон" для этой должности, по возможности, с указанием диапазона заработных плат.
Интересует ТОЛЬКО удаленная работа, желательная тайм-зона - +/- 3 часа от времени {ТАЙМЗОНА}.
Ищи глубоко, можешь использовать fetch, активно используй парсинг контента, думай очень тщательно.
Создай команду агентов и суб-агентов для выполнения задачи, создай основного агента-менеджера, запускай подзадачи, агентов и суб-агентов параллельно, создавай и модифицируй их на ходу по необходимости.
Результат сохрани в файле {ФАЙЛ}.md 
```
[Пример результата](./examples/Audit.md)

## 2. Анализ существующего профиля на LinkedIn
### Prompt:
```
Распарси мой профиль {URL профиля}.
Проанализируй в нём абсолютно всё. 
Насколько он соответствует текущим тенденциям поиска? Насколько релевантен? Каковы шансы, что он будет замечен и оценен достаточно высоко? Что нужно исправить, убрать, дополнить и так далее.
Работай по ранее сформулированным принципам, отчет сохрани в {НАЗВАНИЕ}.md.
```

**ВАЖНО!**
 *Возможна ситуация, когда ИИ не справится с парсингом профиля (из-за защиты от ботов на сайте).
В таком случае лучше выгрузить профиль в PDF и проводить анализ уже на основе PDF-версии.*

[Пример результата](./examples/Report.md)

## 3. Уточнение по позициям в резюме
### Prompt:
```
Задавай вопросы по позциям, чтобы уточнить и  дополнить их необходимыми данными.
Результат сохрани в тот же .md-файл.
```

## 4. Внесение изменений
### Prompt:
```
Внеси правки по плану, учти всё обсужденное, подготовь вариант нового резюме и сопоставь его с шаблоном вакансии.
```

## 5. Подготовка результата
### Prompt:
```
Покажи финальный отчет
```
Тут возможны итерационные улучшения до тех пор, пока результат не будет удовлетворительным.
Возможные варианты запросов:
### Prompt:
```
Давай исправим те позиции, оценка которых меньше меньше 8/10.
Задавай вопросы по скиллам, будем дополнять и исправлять.
```
### Prompt:
```
Переведи итоговый вариант на хороший бизнес-английском
```

---

## 6. Оценка и коррекция результата
### Prompt:
```
Проверь стилистику, грамматику и орфографию.
Проанализируй результаты и определи "коэффициент ИИ" - насколько вероятно, что резюме написано ИИ? Проверь по всем известным маркетам маркеры - длинные тире, смайлики, языковые конструкции.
Сравни результат с написанием резюме вручную для позиции {ВАКАНСИЯ}
Оцени, как высоко будет ранжировано такое ИИ-поисковиками и анализами резюме (например, ATS-системами)?
```
В результате может потребоваться корректировка по отдельным позициям.
### Prompt:
```
Давай заполним то, что меньше 8/10.
Задавай вопросы по скиллам, будем дополнять.
```
Опционально
### Prompt:
```
Проведи итоговый комплексный анализ всех результатов на корректность и на консистентность.
```

```
Создай CV и резюме в формате .md.
Создай шаблон Cover Letter в .md, который будет использоваться для создания адресных Cover Letters.
```
